# KSP Launch Sites Mod Builder

Automatisierter Workflow zum Erstellen von historischen Launch Sites für Kerbal Space Program.

## Setup
1. Hugging Face Account erstellen und API Token holen (Read).
2. Token in GitHub Repo unter Settings -> Secrets -> Actions als `HF_TOKEN` eintragen.

## Nutzung
Gehe in GitHub auf den "Actions" Tab und starte manuell:
- **Launch Site Bildersuche**: Lädt historische Referenzbilder herunter.
- **Generiere Blender-Skript**: Holt Daten von Wikipedia und lässt Qwen ein Blender-Grundgerüst bauen.

## Kosten
100% Kostenlos (GitHub Actions Free Tier + Hugging Face Free Tier).
